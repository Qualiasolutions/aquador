**Implementation:**

1. **Add getProductById import** (after line 7):
```typescript
import { getProductById } from '@/lib/product-service';
```

2. **Replace direct JSON.parse with reconstruction logic** (lines 403-413):

Replace:
```typescript
if (metadata.items) {
  // Standard cart checkout — items stored as JSON array
  try {
    items = JSON.parse(metadata.items);
  } catch (parseError) {
    Sentry.captureException(parseError, {
      tags: { action: 'parse_order_items' },
      extra: { sessionId: session.id },
    });
    console.error('Failed to parse order items:', parseError);
  }
}
```

With:
```typescript
if (metadata.items) {
  // Standard cart checkout — parse shortened metadata (pid, vid, qty)
  // Reconstruct full item details from product catalog
  try {
    const shortItems = JSON.parse(metadata.items) as Array<{
      pid: string;
      vid: string;
      qty: number;
    }>;

    for (const shortItem of shortItems) {
      const product = getProductById(shortItem.pid);
      if (product) {
        items.push({
          name: product.name,
          quantity: shortItem.qty,
          price: product.salePrice || product.price,
          productType: product.type,
        });
      }
    }
  } catch (parseError) {
    Sentry.captureException(parseError, {
      tags: { action: 'parse_cart_items' },
      extra: { sessionId: session.id },
    });
    console.error('Failed to parse cart items metadata:', parseError);
  }
}
```

**Why this exact pattern:**
- Uses `getProductById(shortItem.pid)` like session-details API (proven working)
- Reconstructs `{ name, quantity, price, productType }` matching OrderItem interface
- Preserves Sentry error tracking
- Handles missing products gracefully (skip if not found)
- Uses `product.salePrice || product.price` for correct price (handles sales)

**Do NOT:**
- Change custom perfume branch (line 414+) — it already works correctly
- Change OrderItem interface — it's already correct
- Add any validation here — checkout API already validates before creating session
