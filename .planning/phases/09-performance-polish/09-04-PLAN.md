**Fix 1: Eliminate N+1 in getRelatedProducts (lines 125-144)**

Current implementation:
```typescript
export async function getRelatedProducts(productId: string, count: number = 4): Promise<Product[]> {
  const product = await getProductById(productId);  // Query 1
  if (!product) return [];

  const supabase = createPublicClient();
  const { data, error } = await supabase
    .from('products')
    .select('*')
    .eq('category', product.category)  // Query 2 - uses result from Query 1
    .eq('is_active', true)
    .neq('id', productId)
    .limit(count);
  // ...
}
```

**Solution:** Accept category as parameter OR do single join query:

export async function getRelatedProducts(
