Add global CSS utilities for animation performance and accessibility:

**src/app/globals.css:**

Add after existing custom properties (around line 150+):

```css
/* ==========================================================================
   ANIMATION PERFORMANCE UTILITIES
   ========================================================================== */

/* Optimize for animations - apply to elements that will animate */
.optimize-animation {
  will-change: transform, opacity;
  backface-visibility: hidden;
  -webkit-font-smoothing: antialiased;
}

/* Remove optimization after animation completes */
.optimize-animation-done {
  will-change: auto;
}

/* Force GPU acceleration for smooth transforms */
.gpu-accelerated {
  transform: translateZ(0);
  will-change: transform;
}

/* Smooth scroll behavior (respects reduced motion) */
html {
  scroll-behavior: smooth;
}

/* ==========================================================================
   REDUCED MOTION SUPPORT
   ========================================================================== */

/* Disable all animations for users who prefer reduced motion */
@media (prefers-reduced-motion: reduce) {
  *,
  *::before,
  *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
    scroll-behavior: auto !important;
  }

  /* Prevent Framer Motion from animating */
  [data-framer-appear-animation] {
    animation: none !important;
  }
}

/* ==========================================================================
   HOVER PERFORMANCE OPTIMIZATION
   ========================================================================== */

/* Only apply will-change during hover, not always */
@media (hover: hover) {
  .hover-optimized:hover {
    will-change: transform;
  }
}

/* Touch devices - no hover optimization needed */
@media (hover: none) {
  .hover-optimized {
    will-change: auto;
  }
}
```
